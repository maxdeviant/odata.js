inlineCount(option)

Include a count of all the items in the data set in the response. If used in conjunction with filter, the filtering will take place before the count. Truthy values (e.g., true, 1) as the argument will set the value to 'allpages'. Falsy values (e.g., false, 0) as the argument will set the value to 'none'. Not passing in a parameter will set the value to 'allpages'.
Parameters:
Name Type Description
option Boolean | Integer | String Possible values are 'allpages' and 'none'.
